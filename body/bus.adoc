=== USB
**泛用序列匯流排**(USB, Universal Serial Bus) 是目前最流行的接頭型式，
隨著規格不斷地擴充與演進，大有一統個人電腦周邊設備介面的趨勢。
主要世代與相應的 GNU/Linux 驅動程式以及規格速度如下：

[square]
* USB 1.0: ohci 1.5Mbps
* USB 1.1: ohci 12Mbps
* USB 2.0: ehci 480Mbps
* USB 3.0: xhci 5000Mbps

==== HID
USB HID 介面的狀態有可能會受鍵盤的鎖定燈號影響：
--
　　root# *usbhid-dump -m VID:PID -es*
--

==== VCP
**虛擬序列埠**(VCP) 介面的 /dev/ttyUSB* 無法以 setserial 設定，而是以 stty 命令：
--
　　root# *stty -a -F /dev/ttyUSB0* \#顯示目前設定值 +
　　root# *stty -F /dev/ttyUSB0 19200 -parenb cs8 -cstopb -ixon* +
　　root# *cat < /dev/ttyUSB0 |xxd -g 1*
--
IMPORTANT: 最好先停用或移除 ModemManager 套件以免 VCP 受到影響。

如果想要將十六進位的字元直接寫入裝置中，可參考下面的例子：
--
　　root# *echo -en _'\x01\x01\xCA\x60\x00\x03\xA9'_ > /dev/_ttyUSB0_*
--
或是借助 xxd -r -p 命令，先進行轉換再輸出：
--
　　root# *echo -n '_01 01 CA 60 00 03 A9_' | xxd -r -p > /dev/_ttyUSB0_*
--

== 作業系統安裝

=== 安裝媒體

.iso 檔案是包含光碟完整內容的映像檔格式(ISO 9660)，或稱之為光碟檔案系統(CDFS)。
除非是從書籍或雜誌取得附贈的實體光碟片，否則一般的 GNU/Linux 發行版都衹會提供
.iso 格式的映像檔，讓使用者自行下載後，再以燒錄機製作出安裝光碟。

ISO 9660 並未使用開頭的 32K 位元組，因此可以加入額外資訊來模擬其他的儲存裝置。
由於 USB 隨身碟容量日增、價格日減，又有可以重覆寫入的方便性，因此大部分的
.iso 映像檔都被製作成支援 USB 開機的功能。也就是說，使用者可以像操作磁碟映像檔(.img, .dd)
一樣，將 .iso 檔案以 *dd* 命令直接寫入 USB 隨身碟，製作出可開機的安裝媒體：
--
　　root> *dd if=_install_.iso of=/dev/_sdb_ bs=4M*
--

=== 安裝介面
使用者在安裝過程中面對的操作環境，通常是文字、圖形、網頁三種介面之一：

[square]
* 文字介面
* 圖形介面
* 網頁介面
 安裝媒體啟動後不提供文字或圖形介面，而是直接建立一個網頁伺服器。
 使用者需要利用另一臺機器，透過系統中的瀏覽器來開啟該網路位址的網頁，
 然後依照網頁中的功能進行安裝。

== 前言

Linux 相容性測試人員是指在資訊系統硬體製造的公司中，
針對所生產的各款機型，進行 GNU/Linux 作業系統的運作測試，
包括系統安裝、驅動程式、執行效能等項目。

本書的目標並不在於涵蓋相容性測試人員需要的所有知識，
只希望藉由依照硬體問題分類的編排，能夠加速搜尋問題
發生原因以解決方法的時間，並且提供進一步研究的機會。

限於筆者本身的技術程度，對於實務可行的對策，可惜未能提供充分的理論解釋，
而許多相關的建議方法也無法逐條驗證，衹能附記在旁做為之後類似狀況的參考。
如果讀者手邊的機器已經負有重要的任務，請在應用這些方法前，務必再三查證。

對於已經具備實務經驗而想要深入某個主題的讀者，書末列出了幾本相關的參考書籍，
可以提供各位進一步的探究。
